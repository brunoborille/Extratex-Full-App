HOW TO USE
=============================================================================

1. Navigate to the "Formulas" tab (Settings icon)
2. Create a new client or select an existing one
3. Click "Create Formula" or use a template
4. Define formula details (name, category, type, description)
5. Write the formula expression using variables and functions
6. Add variable definitions
7. Test with sample inputs
8. Save when validation passes

=============================================================================
EXAMPLE USE CASES
=============================================================================

Client A - Premium Terms:
  Formula: (assay * 0.97 - 1) / 100 * dryWeight
  Pays 97% with 1% minimum deduction

Client B - Standard Terms:
  Formula: (assay * 0.965 - 1.5) / 100 * dryWeight
  Pays 96.5% with 1.5% minimum deduction

Client C - Gold Threshold:
  Formula: IF(assay > 0.5, (assay * dryWeight / 31.1035) * 0.95, 0)
  Only pays if gold content exceeds 0.5g/MT

Client D - Volume-based Charges:
  Formula: MIN(dryWeight * 80, 5000) + IF(dryWeight > 100, (dryWeight - 100) * 20, 0)
  Base charge with volume premium
